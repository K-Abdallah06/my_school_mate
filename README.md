📝 About the Project
The manual method of processing student results is often associated with calculation errors, time wastage, and the risk of data loss. The Student Result Management System (my_school_mate) addresses these problems by providing a seamless, automated platform for recording scores and computing grades.

This system ensures 100% accuracy in grading, provides instant access to a student's academic history, and incorporates security features such as password encryption and digital signatures to maintain data integrity.

✨ Key Features
Automated Computation: Automatically calculates Continuous Assessment (CA) scores, Examination scores, and terminal averages.

Report Generation: Generates and prints error-free, terminal report sheets for students.

Role-Based Access Control: Secure, customized dashboards for Administrators, Teachers, and Students.

Data Security: Features password encryption and digital signatures to keep school and student data safe.

Instant History: Provides immediate, centralized access to student academic records.

👥 User Modules
Administrator(pincipal): Manages core school settings, oversees user accounts, and approves final results.

Teacher: Dedicated portal for seamlessly entering and updating CA and Exam scores for their respective classes.

Student: Personal portal to view academic history and print terminal report cards.

🛠️ Built With
Frontend: HTML5, CSS3 AND JAVASCRIPT

Backend: PHP (Hypertext Preprocessor)

Database: MySQL

Methodology: Waterfall Model
